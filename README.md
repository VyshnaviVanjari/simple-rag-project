# SIMPLE RAG PROJECT

This project implements a basic Retrieval-Augmented Generation (RAG) system using Python.

## Features:
Load multiple documents
Split into chunks
Generate Embeddings
Store in FAISS vector database
Retrieve relevant context
Generate answers using LLM

## Setup
pip install -r requirements.txt

Add API key in .env:
OPENAI_API_KEY=your_key_here

## Run
python main.py
